# Feature Implementation

Analyze the project first.

Identify all affected files.

Explain architecture.

Wait for approval.

Then implement feature fully.

---

# Bug Fix

Find root cause.

Explain issue.

Fix only required files.

Do not refactor unrelated code.

---

# Refactor

Find duplicate logic.

Consolidate into reusable functions/components.

Keep behavior identical.

---

# Code Review

Scan entire project.

Find:
- bugs
- inefficiencies
- missing imports
- duplicated logic

Rank issues by severity.
